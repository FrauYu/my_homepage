Framework

The basic structure of the directory is listed below:

my_homepage/

app/

_init_.py

_routes.py

templates/

base.html

home.html

environment.html

framework.html

summary.html

reference.html

static/

photo.png

venv/

homepage.py

Before running the application, the environment variable 'FLASK_APP' should be set:

(venv) $ export FLASK_APP=homepage.py

Then the web application can be run eventually by the following command:

(venv) $ flask run
